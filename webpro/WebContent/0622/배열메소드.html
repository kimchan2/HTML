1. 배열

- a = [11, 100, 180, 53, 22, 230, 450]
- reverse : 배열을 거꾸로 뒤집기
- sort() : 배열을 거꾸로 뒤집기
- sort(function(a, b)){} : 오름차순 정렬 => 11, 22, 53, 100, 180, 230, 450
- sort(function(a, b)){} : 내림차순 정렬 => 450, 230, 180, 100, 53, 22, 11

2. 배열

- a = ["사과", "포도", "수박", "참외", "파인애플", "망고", "자몽"]
- slice(start, end) : start ~ end-1까지의 추출
- splice(index, n, a, b, c, ...) : 수정 - 삽입, 추가, 삭제, 삭제 후 추가
- splice(3, 0, "메론") : 3번째 위치에 메론을 삽입
- splice(3, 1, "메론") : 3번째 위치에 1개를 지우고 메론을 삽입
- splice(2, 2) : 2번째 위치에 2개를 삭제

3. 배열

- a = [11, 100, 180, 53, 22, 230, 450]
- a2 = ["사과", "포도", "수박", "참외", "파인애플", "망고", "자몽"]
- a.concat(a2) : a와 a2가 합쳐져서 새로운 배열을 생성
- a.join() : 배열 a를 하나의 문자열로 변환
- a2.join() : 배열 a2를 하나의 문자열로 변환
